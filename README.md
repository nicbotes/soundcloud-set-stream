Soundcloud Set Stripper
=========

Use
-------
if this is the proper url
'https://soundcloud.com/boebzy/sets/stereodose'

then append the url
`/boebzy/sets/stereodose`

to `http://0.0.0.0:4567/strip_set`

-----
`http://0.0.0.0:4567/strip_set/boebzy/sets/stereodose`
